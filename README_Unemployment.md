# 🇮🇳 India Unemployment Analysis Dashboard

## What it does
An interactive dashboard to explore how unemployment rates changed across
Indian states, especially during the COVID-19 period.

## Features
- Select any state and see its unemployment trend over time
- KPI cards: average, highest, and lowest unemployment rate for that state
- A simple "COVID impact level" indicator (Low / Moderate / Severe)
- Overall India-wide unemployment trend
- Bar chart ranking all states by average unemployment rate

## How it works
- **Dataset**: Unemployment in India.csv
- **Tools**: Python, Pandas, Plotly, Streamlit

## How to run
```
cd Unemployment_Analysis
streamlit run app.py
```
This opens automatically in your browser.
